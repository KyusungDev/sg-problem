## Available Scripts

In the project directory, you can run:

### `yarn start`

개발 모드에서 앱을 실행합니다. <br />
[http://localhost:3000](http://localhost:3000)을 열어 브라우저에서 실행합니다.

### `yarn test`

테스트 runner를 실행합니다.

### `yarn build`

프로덕션용 앱을`build` 폴더에 빌드합니다. <br />
